' ===== Relationships =====

' A supplier supplies many items; each item has one designated supplier
Supplier "1" -- "0..*" InventoryItem : supplies >

' Each item has a demand forecast
InventoryItem "1" -- "1" DemandForecast : has >

' Orders are processed based on demand forecasts
DemandForecast "1" -- "0..*" Order : drives >

' Shipments are made based on order references
Order "1" -- "0..*" Shipment : results in >

' System orchestrates the core entities
InventoryOptimizationSystem ..> InventoryItem : manages
InventoryOptimizationSystem ..> DemandForecast : forecasts
InventoryOptimizationSystem ..> Order : processes
InventoryOptimizationSystem ..> Supplier : evaluates
InventoryOptimizationSystem ..> Shipment : tracks
